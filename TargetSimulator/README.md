# TargetSimulator
目標模擬ライブラリ

## 動作環境

* Ubuntu16.04 32bit
* Qt5.5
* Qt3D


## 開発環境

以下の手順による。

``` bash
$ sudo apt install qt5-default
$ sudo apt install qtcreator
$ sudo apt install qtdeclarative5-dev
$ sudo apt install qt3d5-dev
```
